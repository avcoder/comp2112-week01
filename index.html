Summary


  • JS has matured due to AJAX, Gmail/Google Maps, JQuery, Node, Chrome (V8), ES6,
  • Web development has changed pretty quickly within the last few years. More and more logic moves from the server side to the client side. JS moved to the server (i.e. Node) The server part can be reduced to just serve static files and data over REST services. Web technology also moved to the desktop (i.e. Electron)
  • Learned some ES6 updates to the JS language
